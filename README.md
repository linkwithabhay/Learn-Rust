# Learn-Rust

Learn Rust with me.

### How to install Rust?

Go to the [Rust Installation Tool page](https://www.rust-lang.org/tools/install) and install it in your system.

| Version Used                   |
| ------------------------------ |
| v1.85.0 (d73d2caf9 2024-12-31) |

### How to run it?

Open the CLI inside the root directory and run below command:

```
cargo run
```
